The Issue We're Solving

Current confusion: The dashboard mixes two different time periods without making it clear:

  • Row 1 cards (Total, City Count, County Count) show 2025 data only
  • Row 2 cards (Membership/Other Lobbying) show all-time cumulative data

Goal: Make it immediately obvious which metrics are current year vs historical totals, so users don't accidentally compare 2025 counts with all-time spending.

How to Choose

Pick Option 1 if you want clear separation between current vs historical data

Pick Option 2 if you want to easily compare city vs county lobbying

Pick Option 3 if you like the current layout and just want better labeling
